Plantilla:Cuenta regresiva/uso

De: La Frikipedia, la enciclopedia extremadamente seria.

 Este anexo es una documentación acerca del uso apropiado de la plantilla Cuenta regresiva.
Los cambios hechos aquí serán transcluidos a esta página. Puedes continuar editando esta sección si así lo preferieres.

Esta plantilla calcula el tiempo que falta para una determinada fecha, pudiendo usarse para indicar una cuenta regresiva a un cierto evento.

Uso

{{Cuenta regresiva
|evento = 
|año = 
|mes = 
|día = 
|hora = 
|minutos = 
|segundos = 
|tiempo = 
|duración = 
|mensaje inicio = 
|mensaje final = 
|color = 
}}

Parámetros

Obligatorios
Ninguno
Opcionales
evento
Nombre del evento
año
Año en que ocurre el evento, sin espacios, comas o puntos separando los miles.
mes
Mes en formato numérico. Enero es 1; febrero, 2; etc.
día
Día del mes, también en formato numérico.
hora
Hora del evento. Siendo un número entre 0 y 23.
tiempo
En vez de dar la fecha se puede dar un tiempo específico en segundos, siendo el 1º de enero de 1970 a las 00:00 hs el segundo 0.
duración
Tiempo en segundos que durará el evento.
mensaje inicio
Mensaje que se visualizará durante el evento.
mensaje final
Mensaje que se visualizará luego de que el evento haya finalizado.
color
Se puede especificar un color para la cuenta atrás, por defecto es rojo.

Ejemplos de uso

{{Cuenta regresiva
|evento = el comienzo del siglo C
|año = 9901
|mes = 1
|día = 1
|hora = 0
|minutos = 0
|segundos = 0
}}
Resulta en

Faltan 2879683 días 15 horas para el comienzo del siglo C. 

{{Cuenta regresiva
|evento = El [[día de pi]]
|tiempo = 3141592654
|color = green
}}
Resulta en

Faltan 19306 días 15 horas para el día de pi. 

{{Cuenta regresiva
|año = 2000
}}
Resulta en

El evento ha finalizado. 

{{Cuenta regresiva
|evento = el siglo XXI
|inicio = 2000
|mes = 12
|día = 31
|duración = 3600
|mensaje inicio = ¡Falta un día para el siglo XXI!
|mensaje final = ¡Bienvenido siglo XXI!
}}
Resulta en

El siglo XXI ha finalizado.
