Plantilla:Frikibox-2

De: La Frikipedia, la enciclopedia extremadamente seria.

id1 info id2

Uso

{{Frikibox-2
|align    = alineación
|border-c = color de la caja
|border-s = grosor de la caja
|id1-c    = id-1 color de fondo
|id1-s    = id-1 tamaño del texto
|id1-fc   = id-1 color de fuente
|id2-c    = id-2 color de fondo
|id2-s    = id-2 tamaño del texto
|id2-fc   = id-2 color de fuente
|info-c   = info color de fondo
|info-s   = info tamaño del texto
|info-fc  = info color de fuente
|id1      = id-1 texto o imagen
|id2      = id-2 texto o imagen
|info     = contenido de la caja "info"
}}

Ejemplo


{{Frikibox-2
|align    = left
|border-c = green
|border-s = 3
|id1-c    = #854555
|id1-s    = 10
|id1-fc   = #042
|id2-c    = red
|id2-s    = 9
|id2-fc   = #555
|info-c   = #EAEAE8
|info-s   = 5
|info-fc  = #E58B37
|id1      = ID-1
|id2      = ID-2
|info     = contenido de la caja "info"
}}


ID-1 contenido de la caja "info" ID-2
