Plantilla exclusiva para el uso del proyecto Efemérides.

Esta plantilla consta de dos estilos:

Primer estilo: Calendario (por defecto)

Para usar este estilo, hacer la inserción de esta forma: {{Efemérides-Mes}} o {{Efemérides-Mes|1}}

Parámetros

Introducir estos parámetros luego del estilo.

  • Mes: Define el mes que se mostrará. Valor por defecto: "{{CURRENTMONTHNAME}}".
  • Barra: Define si aparece una barra acompañando al nombre del mes. Valor aceptado: "1" - para mostrar la barra.
  • Barra-Color: Define el color de la barra. Valor por defecto: "#ccccff".
  • Alineación: Alinea el calendario al gusto (left, center, right). Valor por defecto: "center".

Ejemplos de uso

  • Si se introduce un valor erróneo: la plantilla trabajará con el mes actual.

{{Efemérides-Mes|Mes=jul}}

  • Si no se especifica un mes en particular: la plantilla tomará el actual como base.

{{Efemérides-Mes}}

  • Si se requiere, la plantilla puede añadir una barra -el color puede ser cambiado- en el nombre del mes.

{{Efemérides-Mes|Mes=Mayo|Barra=1|Barra-Color=steelblue}}

Segundo estilo: Corrido

Para usar este estilo, hacer la inserción de esta forma: {{Efemérides-Mes|2}}


    

                                                           

                                                  


Parámetros

  • Mes: Define el mes que se mostrará. Valor por defecto: "{{CURRENTMONTHNAME}}".
  • Espacio: Espacio entre cada número. Valor por defecto: "3".
  • Color: Define el color de los número. Color por defecto: Negro.
