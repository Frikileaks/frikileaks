Es una de las letras del avesedario frikipédico. Que, a diferencia de las otras letras, esta no se usa muy a menudo por lo difícil que es marcarla con el teclado (Era algo así como Alt+f4) pero es increíblemente fácil de pronunciar, /→/. Se parece al sonido que se hace cuando apuntas algo con el dedo.

Datos varios sobre la →

Uso tipico de la →

Primero que nada es una flecha, y es una de las letras menos usadas en la tierra ya que solo se usa para decir una que otra boludés en remeras o paredes, o para indicar cosas sin usar el dedo, ejemplos: "Estoy con un gilopollas →", "¡Mira! Una ramera →" o "Usted esta aquí →". A diferencia de lo que mucha Any key) en lugar de simplemente apuntar con el dedo que dedo. Algunas cosas que poca gente sabe sobre la → es que aparece en el teclado, aunque casi nunca se escribe.


Reglas Hortograficas sobre la →

  • La → solo se puede escribir sola, no se acompaña de ninguna otra clase de letra. Excepto cuando se acompañe de una ←, o de otras letras para hacer caritas/emoticonos.
  • La → se acentua en la → pero no lleba tilde. A menos que seas francés, entonces lleva tilde en la ←
  • Cualquier palabra puede ser proseguida por la → pero casi ninguna queda bien.
  • La → puede rotarse para conveniencia. En ángulos de 90°, 180°, 260°, 360°, 720° y 20.14.000 grados. También diagonales en una angulo de 45°, 135°, 12323023 grados.
  • La → tambien se puede escribir sola sin ninguna referencia. A esto se le conoce como oración con → tácita.
  • En una oración, se puede eliminar el sujeto, convertir la → en objeto y poner el verbo en pasado recontraparticipio. A esto se le conoce como una estupidez gramatical.
  • No se debe colocar → juntas puesto que es redundante y es gramaticalmente ilegal.
