Plantilla:Otros usos/uso

De: La Frikipedia, la enciclopedia extremadamente seria.

 Este anexo es una documentación acerca del uso apropiado de la plantilla Otros usos.
Los cambios hechos aquí serán transcluidos a esta página. Puedes continuar editando esta sección si así lo preferieres.

Esta plantilla sirve para enlazar con otros artículos que tengan una definición diferente y posean un mismo título.


Si el término tiene múltiples usos, escribe
{{Otros usos}}


Si sólo tiene otro término, se usa
{{Otros usos|Caca}}

y te saldrá:
Para otros usos de este término/definición/cosa, véase Caca.


En caso de querer mencionar en que consiste el otro término, se escribe
{{Otros usos|Truño|truño como expresión}}

y obtendrás:
Para truño como expresión, véase Truño.


Pero si lo que quieres es mencionar de que trata este artículo y diferenciarlo de la otra definición, agregas
{{Otros usos|Truño|truño como espresión|la caca como materia biológica}}

y tendrás:
Este artículo trata sobre la caca como materia biológica, para truño como expresión, véase Truño.
