Tecla situada entre Ctrl y Alt, para separarlas y así facilitar la combinación Ctrl+Alt+Supr a quienes tienen dedos gruesos o carecen de pulgar opuesto. Inventada por Microsoft para introducir publicidad subliminal, es el primer intento documentado de spam por hardware.

Desde el incidente con el Niño loco alemán, se la fabrica con un material más resistente que el del resto de las teclas para que sea lo único que sobreviva la furia de determinados usuarios.

La tecla Windows (FPS) y minimice el juego desplegando el menú Inicio. De nada servirá querer volver a maximizar el juego, porque el SO está preparado para darle prioridad baja, dándole tiempo a los NPC's más bizcos de efectuar un headshot con una granada de humo.

Uso en GNU/Linux

A pesar de esto, el objetivo principal es hacer creer a los ignorantes que el sistema operativo Windows que vino preinstalado en la computadora es la única opción y que no se puede quitar porque el Teclado dejaría de funcionar. De hecho, los diferentes escritorios como GNOME o KDE han debido imitar su funcionamiento para que los que nunca usaron un sistema GNU/Linux no rompieran el teclado al grito de "Esta mierda no anda"; en susodichos sistemas la tecla deja de utilizar el original nombre de "Tecla de Windows" para obtener el no menos original nombre de "Vade retro Satanas" "Mod4". Entre las utilidades que trae por defecto en Ubuntu 10.10 encontramos:

[Mod4] + [E] = Mostrar el panel de escritorios.

[Mod4] + [W] = Mostrar el selector de ventanas (como [Ctrl] + [Alt] + [Tab], pero mas "PRO")

[Mod4] + [Ctrl] + [Alt] + {Símbolo de Batman} = Fractura multiple de muñeca, un minimo de dos dedos y medio dislocados, ruptura de ligamentos y peste bubónica.
