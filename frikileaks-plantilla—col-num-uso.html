Plantilla:Col-num/uso

De: La Frikipedia, la enciclopedia extremadamente seria.

 Este anexo es una documentación acerca del uso apropiado de la plantilla Col-num.
Los cambios hechos aquí serán transcluidos a esta página. Puedes continuar editando esta sección si así lo preferieres.

Esta plantilla solo funciona con Mozilla Firefox.

Uso

{{Col-num|NÚMERO DE COLUMNAS|TEXTO PARA HACERLO COLUMNAS}}

Ejemplo: Si quieres hacer tres columnas a...

  1. a
  2. b
  3. c
  4. d
  5. e
  6. f

...escribe esto... 

{{col-num|3|
#a
#b
#c
#d
#e
#f
}}

...y saldrá...

  1. a
  2. b
  3. c
  4. d
  5. e
  6. f
