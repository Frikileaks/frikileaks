 Este anexo es una documentación acerca del uso apropiado de la plantilla Nowrap.
Los cambios hechos aquí serán transcluidos a esta página. Puedes continuar editando esta sección si así lo preferieres.

Uso

Esta plantilla evita los saltos de línea en textos que usen espacios.

{{nowrap|<texto>}}

Ejemplo

El siguiente ejemplo impide que «10» y «metros» se muestren en líneas distintas:

{{nowrap|10 metros}}

Handling interpreted characters

Para un texto que incluya un signo de igual "=", precede el texto 1=. Por ejemplo:
{{nowrap|1=2 + 2 = 4}}
Which renders:
2 + 2 = 4
Para un texto que incluya una barra vertical "|", escape the bar(s) with &#124; o {{!}}. Por ejemplo:
{{nowrap|&#124;2&#124; < 3}}
Which renders:
|2| < 3
