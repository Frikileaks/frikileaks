Uso

{{purgar}} produce un enlace que al ser pulsado limpia la cache del servidor y reconstruye la página. De manera predeterminada, muestra el texto «purgar». Si deseas cambiarlo, añádele un parámetro a la plantilla: {{purgar}} para refrescar la página actual. {{purgar|página}} para refrescar otra página. {{purgar|página|página=texto}} refrescar otra una página bajo un texto diferente de «purgar».

  • Ejemplo {{purgar|Recargar esta página}} → Recargar esta página
